## Tables

### `pdf_documents`

Stores generated or uploaded PDF document metadata and binary content.

| Column | Type | Required | Notes |
| --- | --- | --- | --- |
| `id` | `BIGINT` | Yes | Identity primary key |
| `file_path` | `VARCHAR(255)` | Yes |  |
| `original_file_name` | `VARCHAR(255)` | Yes |  |
| `created_at` | `TIMESTAMP` | No | Defaults to `CURRENT_TIMESTAMP`; Creation timestamp |

## SQL

```sql
CREATE TABLE pdf_documents (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY,
    file_path VARCHAR(255) NOT NULL,
    original_file_name VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    PRIMARY KEY (id)
);
```
